# Plugins

Community plugin system for Safwa.

## Overview

Safwa supports community-built plugins to extend functionality. This directory contains:

- Plugin API specification
- Plugin development guide
- Example/template plugins
- Plugin registry format

## For Plugin Developers

Community plugins live in their own repositories. This space provides the SDK, docs, and templates to get you started building plugins for Safwa.

## Plugin Ideas

- Custom integrations (Spotify, Todoist, Notion import, etc.)
- Additional AI providers
- Hardware integrations (smart home, wearables)
- Custom dashboard widgets
- Import/export adapters
- Notification channels (Telegram, Matrix, etc.)
